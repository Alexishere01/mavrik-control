#           Differential throttle disabled

# KEY FINDINGS:
#
# 1. Differential throttle provides ZERO yaw torque in MAVRIK
#    Proof: v3/v4/v5 changed throttle limit from ±0.03 to ±0.06 to ±0.02
#    Result: identical 81° yaw peak in all three → throttle does nothing
#    Root cause: MAVRIK ignores Cl polynomial in custom component thrusts blocks
#
# 2. Differential flaps (vectored tilt) IS the yaw actuator
#    The forward tilt motors create a horizontal force couple when tilted asymmetrically
#    At flaps=0.99 (hover), even small asymmetry creates meaningful yaw torque
#
# 3. Proportional-only flaps control causes limit-cycle oscillation (v1-v5)
#    Fix: Route full cascade PID (att→rate→flaps) to provide rate damping (v6+)
#
# 4. Rate PID derivative (kd) above 0.004 amplifies sensor noise
#    v9 kd=0.006 → flaps chatter ±0.15 at ~10Hz → 13° pitch disturbance
#    v8 kd=0.004 → clean, no chatter
#
# 5. Flaps authority limit: ±0.12 is the sweet spot
#    Lower (±0.06) = too sluggish, higher (±0.15) = pitch disturbance from tilt
#    At ±0.12, tilted motor at 78° → 97% lift retained, ~25 ft·lbf yaw torque
#
# 6. Roll-to-yaw coupling is the dominant disturbance
#    5° roll command creates sustained yaw drift that the controller must fight
#    This is an inherent property of the vehicle, not a controller bug
